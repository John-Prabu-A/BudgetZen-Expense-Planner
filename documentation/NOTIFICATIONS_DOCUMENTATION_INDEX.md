## üéì Key Concepts from Documentation

### Preference
A user setting (boolean or string) stored in SecureStore and accessible via context.

### Service Integration
Services check preferences before sending notifications - e.g., `if (!budgetAlerts) return;`

### Data Flow
User Action ‚Üí State Update ‚Üí SecureStore ‚Üí Console Logged ‚Üí useEffect ‚Üí Next Session

### Master Switch
Push Notifications is the master switch - disabling it disables all push notifications.

All times stored as "HH:MM AM/PM" and converted from input formats automatically.
