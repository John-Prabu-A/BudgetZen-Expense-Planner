## Key Insights Users Can Derive

### Income Analysis:
1. **Primary Income Source**: Which account receives most income?
2. **Income Stability**: How often do income transactions occur?
3. **Income Consistency**: Is average per transaction reliable?
4. **Income Peaks**: When do peak income days happen?
5. **Income Patterns**: Any recurring schedule (weekly/bi-weekly)?

### Expense Analysis:
1. **Major Expense Accounts**: Which account spends the most?
2. **Spending Frequency**: How many days per month have expenses?
3. **Average Spending**: What's typical expense per transaction?
4. **Spending Spikes**: When are unusually high expense days?
5. **Spending Patterns**: Any recurring expense patterns?
